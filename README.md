# Atelier Nova

Landing page estática criada em HTML, CSS e JavaScript.

## Estrutura

- `index.html`: estrutura principal da página
- `styles.css`: identidade visual, layout responsivo e animações
- `script.js`: revelação progressiva das seções ao rolar

## Como abrir

Abra `index.html` diretamente no navegador.

## Próximos passos

- publicar em GitHub Pages, Vercel ou Netlify
- substituir os textos pela marca final
- integrar formulário ou analytics se necessário
