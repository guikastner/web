# Atelier Nova

Landing page criada com HTML, Tailwind CSS, Vite e JavaScript.

## Estrutura

- `index.html`: estrutura principal da página
- `styles.css`: entrada Tailwind com tokens, camadas base/componentes e animações
- `script.js`: revelação progressiva das seções ao rolar
- `vite.config.js`: configuração Vite com plugin oficial do Tailwind CSS

## Como rodar

Instale as dependências:

```bash
npm install
```

Rode em desenvolvimento:

```bash
npm run dev
```

Gere a versão de produção:

```bash
npm run build
```
